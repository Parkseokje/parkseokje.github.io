Function Statement VS Function Expression

2017-03-15
  • javascript
  • udemy

JavaScript 에만 있는 Function 의 특징이 있다.
바로 ‘1급 XX’ 라는 수식어가 붙는다는 것이다.

1급 시민 (First Class Citizen)
1급 객체 (First Class Object)
1급 함수 (First Class Function)

1급 시민

변수(Variable), 인자(Parameter), 반환값(Return Value)으로 전달할 수 있다.

1급 객체
특정 언어에서 객체를 1급 시민으로 취급한다는 의미이다.

1급 함수
1급 시민이면서, 추가적으로 런타임(Runtime)을 생성할 수 있으며, 익명(Anonymous)으로도 생성이 가능하다.

이번 시간에는 1급 함수 로서 JavaScript Function 의 특징에 대해 알아볼 것이다.

Function 은 두 가지 형태로 분류할 수 있다.

  • Function Statement: 함수 선언문
  • Function Expression: 함수 표현식
